# Perfumería JC

Bienvenido a **Perfumería JC**, tu plataforma online para descubrir y comprar los mejores perfumes y fragancias.

## 🌟 Características

- **Catálogo Completo**: Amplia selección de perfumes y colonias
- **Búsqueda Avanzada**: Encuentra productos por marca, tipo y precio
- **Carrito de Compras**: Gestin fácil de tus compras
- **Información Detallada**: Descripción, notas aromáticas y reseñas
- **Envíos Rápidos**: Entrega a toda el país
- **Atención al Cliente**: Soporte disponible

## 🚀 Tecnología

- HTML5 / CSS3
- JavaScript
- Responsive Design
- SEO Optimizado
